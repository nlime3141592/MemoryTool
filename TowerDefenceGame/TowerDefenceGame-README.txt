[English, 영어]
1. Execute "TowerDefenceGame.exe".
2. Click button "Press to Start".
3. Click button "Tower1". Then, you can place a tower on red cell. (cost=125$)
4. Game is infinite(no exit condition). If you want to quit application, please click the X button on window.

NOTE:
1. If the application window lose focus(click outside of window, etc...), application pauses.
2. Initial having money is 450$. having money automatically increases 1$ per 3 seconds.
3. If tower kill a enemy, then, having money increases 50$.

[Korean, 한국어]
1. "TowerDefenceGame.exe"를 실행하세요.
2. "Press to Start" 버튼을 클릭하세요.
3. "Tower1" 버튼을 클릭하세요. 그러면 빨간색 칸에 타워를 설치할 수 있습니다. (비용=125$)
4. 게임이 무한 지속됩니다(종료 조건 없음). 만약 애플리케이션을 종료하고 싶다면 창의 X 버튼을 클릭하세요.

참고:
1. 만약 애플리케이션 창이 포커스를 잃으면(창 밖 클릭 등..) 애플리케이션은 일시 정지합니다.
2. 초기 소유한 돈은 450$입니다. 소유한 돈은 3초에 1$씩 자동으로 증가합니다.
3. 만약 타워가 적을 처치하면, 소유한 돈이 50$ 증가합니다.
